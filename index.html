Just a year ago, I was researching the latest trends and translating them into large-scale advertising campaigns that would reach millions of people. The more I learned about how digital ads targeted and interacted with the masses, the more I grew interested in the technology behind these giant digital platforms. I wanted to understand the inner-workings of this powerful technology so I began taking online programming classes after work each evening. It was not long before I was hooked.

After implementing control flow to build my first game in the terminal, I knew programming was going to be my new career. Since then I've build several apps at an immersive 12-week bootcamp and worked as a full time Front End Web Developer fixing bugs. You can see some of my projects below, with the latest one being built with React and Redux for maximum Pokemon fun!

I'm thrilled to continue growing as a developer and contribute to an awesome team. If you know of any opportunities, please feel free to contact me at jackyklei@gmail.com
